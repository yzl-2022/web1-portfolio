La description d'une réalisation

Composants du système informatique personnel

Le disque dur Contrairement à la mémoire vive, les données stockées sur cet élément sont permanentes et ne sont pas effacées à l'arrêt de votre ordinateur. C'est donc sur celui-ci que votre système d'exploitation (Linux, MacOs ou Windows), vos logiciels et vos documents sont conservés. C'est donc un espace de stockage permanent, où vous pouvez copier et supprimer des données à volonté.
L'alimentation Cet élément a le rôle d'assurer la fourniture en électricité à tous les composant de votre ordinateur.
Le lecteur & graveur cd/dvd Cet élément vous permet de lire les données qui se trouvent sur les CD et DVD (jeux, logiciels, photos). Dans sa fonction graveur, en plus de lire les données, il permet de stocker des fichiers sur des CD et des DVD (musique, film, photos, sauvegardes), afin de libérer de l'espace sur votre disque dur.
La carte son Permet de gérer les entrées et les sorties de votre ordinateur. Elle comporte un processeur chargé de tous les traitements numériques.
La carte graphique Permet de traiter et d'afficher sur un écran les données provenant de votre ordinateur. Les cartes graphiques récentes disposent de processeurs spécialisés dans la manipulation de données en 3D. (Jeux vidéos)
La carte mère C'est elle qui accueille l'ensemble des composants internes de votre ordinateur (processeur, mémoire,...) et gère les différentes interfaces avec vos périphériques: prise pour les éléments internes et ports USB pour les périphériques externes.
RAM (mémoire vive) Partenaire de travail du processeur, stocke temporairement les données à traiter par le processeur. Ainsi, plus il y a de grade capacité, plus il est possible d'y conserver des données temporairement (ce qui évite d'accéder au disque dur qui est plus lent). Elle est vidée à chaque arrêt ou redémarrage de votre ordinateur.
Le processeur Souvent qualifié de cerveau de l'ordinateur, il permet de manipuler et de traiter les données qui lui sont fournies. Sa puissance a une influence sur la vitesse d'exécution de vos logiciels et des opérations que vous effectuez.

La sécurité du web - malware

Modes de sécurité Description
Vers Ce programme se reproduit dans le but de ralentir ou de perturber les opérations réseau.
Logiciel publicitaire Il distrubue lors de téléchargement de logiciels en ligne et affiche de la publicité dans une fenêtre contextuelle.
Logiciel espion Il collecte des informations sur l'utilisateur et les envoyer vers une autre entité, sans le consentement de l'utilisateur.
Faux logiciels de pretection (ransomware) Il bloque l'accès au système infecté et exige le paiement d'une rançon pour que cette restriction soit levée.
Rootkits Ce logiciel est souvent utilisé par les hackers pour obtenir un accès de niveau administrateur à l'ordinateur.

Utiliser git pour gérer les dépôts locaux et distants

La première étape: configurer votre identité

$ git config --global user.name "votre_username"
$ git config --global user.email votre_email@email.com

La deuxième étape: initialiser le dépôt local

$ git init

La troisième étape: cloner le répertoire en local

$ git clone https://github.com/votre_lien.git

La quatrième étape: créer une nouvelle branche

$ git checkout -b nouvelle_branche

une fois les modifications apportées: commit les modifications

$ git add .
$ git commit

À la fin: fusionner avec la branche principale et pousser vers l'origine distante

$ git checkout main
$ git merge nouvelle_branche
$ git push origin main

Installer LAMP dans une machine virtuelle du système d'exploitation Linux

1. installer un hyperviseur pour héberger le système d'exploitation linux (Oracle VirtualBox pour Windows, UTM pour MacOS). télécharger une image d'installation pour Debian 11 (https://www.debian.org/distrib/). créer une nouvelle machine virtuelle.

2. installer Apache2

$ sudo apt-get install apache2 apache2-doc

3. installer PHP

$ sudo apt-get install php

4. installer MariaDB

$ sudo apt-get install -y mariadb-server

5. installer phpMyAdmin

$ sudo apt-get install phpmyadmin
